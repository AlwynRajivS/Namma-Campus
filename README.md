# 🎓 Namma Campus – Family Registration Portal

A smart and modern web-based portal designed for seamless student family registration during academic events like Parent-Teacher Meetings.

---

## ✨ Key Highlights
- 👩‍🏫 Faculty-based student registration system  
- 🛡️ Admin dashboard with real-time insights  
- 📊 Google Sheets as live database  
- ⚡ Fast, responsive, and user-friendly UI  
- 🔗 Powered by Google Apps Script integration  

---

## 🚀 What It Does
- Select students by Department, Year, and Section  
- Register family member details instantly  
- Store and manage data in real-time  
- View complete analytics through admin dashboard  

---

## 🛠️ Tech Stack
- HTML • CSS • JavaScript  
- Google Apps Script  
- Google Sheets  

---

## 🎯 Use Cases
- Parent-Teacher Meetings  
- College Events  
- Student Data Collection  

---

## 🌟 Vision
Simplifying campus data management with smart, scalable, and digital solutions.
